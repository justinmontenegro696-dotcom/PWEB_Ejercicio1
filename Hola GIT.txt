¿Qué diferencia existe entre git pull y git fetch?
git fetch = Descarga los cambios del remoto sin mezclarlos con tu trabajo.
git pull = Descarga los cambios y los mezcla automáticamente con tu rama actual.

¿Para qué sirve el comando git clone?

El comando git clone sirve para copiar un repositorio remoto completo (por ejemplo, de GitHub) a tu computadora.
